# PL: 
Pobieranie danych finansowych z yfinance (yahoo Finance) i przesyłanie do arkuszy Google za pośrednictwem Pythona i n8n, w celu podjęcia decyzji inwestycyjnej (zakup akcji na giełdzie).



zakładki w Google sheets:
1. ANALIZA (analysis) - wyniki modelu + opis wzorów. Cała ewaluacja ma miejsce w tej zakładce, wiersze 1-12, kolumny A - I.
2. surowe_dane (raw data) - dane finansowe spółek, przewalutowane na USD oraz znormalizowane za pomocą funkcji PERCENTRANK
3. surowe_dane_opis (raw data description) - opis danych finansowych oraz interpretacja



# ENG: 
Downloading financial data from yfinance (yahoo Finance) and sending to Google sheets via Python and n8n, in order to make an investment decision (purchase of shares on the stock exchange).

bookmarks in Google sheets:
1. ANALYSIS - model results + description of patterns. All evaluation takes place in this tab, rows 1-12, columns A - I.
2. raw_data (raw data) - financial data of companies, converted into USD and normalized using the PERCENTRANK function
3. raw_data_opis (raw data description) - description of financial data and interpretation
